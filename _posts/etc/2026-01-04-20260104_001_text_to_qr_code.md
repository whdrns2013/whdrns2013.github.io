## 텍스트를 QR 코드로 간단히 공유하기  

### Intro  

텍스트 파일을 QR 코드로 변환하면, 별도의 파일 전송 없이 스마트폰 카메라만으로 내용을 바로 확인할 수 있다. 설정값, 일회성 정보 공유에 특히 유용하니, 이번 기회에 알아보자.  

### QR 코드 생성 도구 설치  

`qrencode` 패키지는 QR코드를 생성할 수 있도록 해주는 툴이다.  

```bash
# Ubuntu
apt install qrencode

# macOS
brew install qrencode
```

### 공유할 텍스트를 담은 파일 생성  

```bash
vi ./sample.txt
```

```plaintext
# sample.txt
this is the sample text file!
```

### 텍스트를 QR 코드로 변환  

아래 명령어를 실행하면, 파일 내용을 담은 QR 코드가 터미널에 출력된다.  

```bash
qrencode -t ansiutf8 < sample.txt
```

![](/assets/images/20260104_001_001.png)  

### 결과 확인  

스마트폰으로 QR 코드를 촬영하면, sample.txt에 작성한 텍스트 내용이 그대로 화면에 표시된다.  

파일 전송 없이도 내용을 즉시 전달할 수 있으며, 특히 설정값 등 복잡하고 긴 텍스트를 옮길 때 적절할 것으로 생각된다.  

![](/assets/images/20260104_001_001.png)  

### 언제 사용하면 좋을까  

- 스마트폰에 긴 텍스트를 옮기기  
- 설정값 옮기기  
- URL QR코드 만들기
