## while 문  

### 정의  

- 조건식이 참(True)인 동안 내부의 문장을 반복해서 실행한다.  
- 반복 횟수가 정해져 있지 않을 때 주로 사용한다.  

### 표현 방법  

```c
while (조건식){
    반복할문장;
}
```

### 예시  

```c
int i = 0;
while (i < 3){
  printf("i = %d", i);
  i++;
}
```

```bash
i = 0
i = 1
i = 2
```
