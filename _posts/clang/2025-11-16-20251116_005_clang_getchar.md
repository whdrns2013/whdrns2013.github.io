## getchar() 함수  


### 정의   

- 표준 입력으로부터 1개의 문자를 입력받는 함수  
- getchar 함수는 scanf 와 달리 **주소 연산자 (`&`)를 사용하지 않는**다.  

### 형식  

```c
#include <stdio.h>
char ch;
ch = getchar();
```

### 예시  

#### 예시  

```c
void get_char(){
    char a;
    a = getchar(); // <-- 입력 A
    printf("a : %c\n", a);
}
```

```bash
# 출력
a : A
```

#### 변수 선언과 동시에 getchar()로 초기화  

```c
void get_char(){
    char b = getchar(); // <-- 입력 C
    printf("b : %c\n", b);
}
```

```bash
# 출력
b : C
```

#### 연속된 문자 뭉치 (=문자열) 을 입력하면 가장 첫 문자만 저장됨  

```c
void get_char(){
    char c = getchar(); // <-- 입력 The Greatest Showman
    printf("c : %c\n", c);
}
```

```bash
# 출력
c : T
```
