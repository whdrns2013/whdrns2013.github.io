## 구조체 멤버 접근하기(참조)  

- `.` 연산자를 이용해 구조체 멤버에 접근할 수 있다.  

```c
struct Circle {
  float x, y;
  float radius;
} c1 = {0.0f, 1.0f, 3.0f};

printf("%f %f %f\n", c1.x, c1.y, c1.radius);
```

```bash
0.000000 1.000000 3.000000
```
