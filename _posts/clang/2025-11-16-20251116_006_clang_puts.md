## puts() 함수  


### 정의   

- 표준 출력에 문자열을 출력하는 함수  
- 출력 후 **자동으로 행이 바뀐**다.  

### 형식  

```c
char* str = "The Greatest Showman";
puts(str);
```

### 예시  

```c
void put_s(){
	char* str = "The Greatest Showman";
	puts(str);
	puts("Kings Man");
}
```

```bash
# 출력
The Greatest Showman
Kings Man
```

> `*str` 과 같이 포인터 변수는 사용하지 않아도 된다.
