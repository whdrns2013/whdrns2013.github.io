## break 문  

### 정의  

- 반복문(for, while, do-while) 이나 switch 문을 강제로 종료할 때 사용하는 제어문이다.  
- 가장 가까운 블록 하나만을 빠져나가는 동작을 수행한다.  

### 표현 방법  

```c
{
    ...반복문 도중...
    break;
}
```

### 예시  

```c
int i = 0;
while (1) {
  if (i == 3){
    break;
  }
  printf("%d\n");
  i++;
}
```

```bash
0
1
2
```
