## continue 문  

### 정의  

- 반복문의 남은 부분을 실행하지 않고, 다음 반복 단계로 넘어가게 한다.  
- while, do-while 에서는 조건식 검사 단계로 넘어간다.  
- for 문에서는 증감식 단계로 넘어간다.  

### 표현 방법  

```c
{
    ...반복문 도중...
    continue;
}
```

### 예시  

```c
for (int i=1; i <= 5; i++){
  if (i == 3){
    continue; // i == 3 일때 건너뜀
  }
  printf("%d", i);
}
```

```bash
1
2
4
5
```
