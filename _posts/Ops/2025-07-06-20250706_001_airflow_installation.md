## Airflow의 설치

### 설치 방법

| 설치 방법 | 설명 |
| --- | --- |
| Using released sources | Apache Airflow의 공식 릴리스 소스 코드를 직접 내려받아 설치하고 환경에 맞게 빌드·구성하는 방법 |
| Using PyPI | Python 패키지 저장소인 PyPI에서 `pip`으로 Airflow를 설치하는 가장 일반적인 방법 |
| Using Production Docker Image | Airflow가 포함된 공식 프로덕션용 Docker 이미지를 사용해 컨테이너 환경에서 빠르게 실행하는 방법 |
| Using Official Airflow Helm Chart | Kubernetes 환경에서 공식 Helm Chart를 이용해 Airflow를 배포하고 관리하는 방법 |
| Using Managed Airflow Services | 클라우드 제공업체의 관리형 Airflow 서비스를 사용해 인프라 운영 부담 없이 Airflow를 이용하는 방법 |
| Using 3rd-party images, charts, deployments | 서드파티가 제공하는 이미지, Helm Chart, 배포 구성을 활용해 Airflow를 설치하는 방법 |

[Installation of Airflow® — Airflow 3.1.8 Documentation](https://airflow.apache.org/docs/apache-airflow/stable/installation/index.html)
